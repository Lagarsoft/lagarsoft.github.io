www.BIMDash.io (Demo) Organizations generate vast amounts of data every day, and our ability to collect that data continues its growth.

Now the challenge is to be able to transform that data into something meaningful and allow people to learn about their performance, their strengths, weaknesses and ultimately to help taking better decisions.

There are a many tools available in the market to collect, analyze and visualize that data, but the real value comes from the ability to make all the pieces to work together, which is specifically the strength of Lagarsoft's team.

Teams focused on performance-driven design, use project data as an asset to the planning process, by comparing the performance of past projects, defining metrics, establishing benchmarks and driving the future work.

We created www.BIMDash.io, a centralized dashboard that enables BIM managers to understand multiple projects in a single dashboard.

Goals:

  • To save BIM Manager's time
  • Centralized, Web-based (updated data in almost real-time)
  • Mobile and Tablet friendly (No need to have Revit or Windows to see data)

Features:

  • Data extracted from Revit API and Journal files
  • Projects Evolution analysis
  • Metrics Driven design
  • Map View
  • Compare projects
  • Benchmarking
  • Time Tracking
  • Compatible with MS Power BI and Tableau
