www.BIMDash.io (Demo) Organizations generate vast amounts of data every day, the challenge is to be able to transform that data into something meaningful and allow people to learn about their performance, their strengths, weaknesses and ultimately to help taking better decisions.

There are many tools available in the market to collect, analyze and visualize that data, but the real value comes from the ability to make all the pieces to work together, which is specifically the strength of Lagarsoft's team.

BIMDash.io is a centralized dashboard that enables BIM managers to understand multiple performance metrics for different projects in a single dashboard.

It can also be easily extended to be used by teams focused on performance-driven design. They can define new metrics used in the planning process, by comparing the performance of past projects, establishing benchmarks and driving future work.

Goals:

  • To save BIM Manager's time
  • Centralized, Web-based (updated data in almost real-time)
  • Mobile and Tablet friendly (No need to have Revit or Windows to see data)

Features:

  • Data extracted from Revit API and Journal files
  • Projects Evolution analysis
  • Metrics Driven design
  • Map View
  • Compare projects
  • Benchmarking
  • Time Tracking
  • Compatible with MS Power BI and Tableau
